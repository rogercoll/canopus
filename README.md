# Directory Encryptor/Decryptor
Golang program that encrypts any type of file given a directory. 
PASSWORD MUST BE 16 CHARACTERS LONG

## Usage
### 1. Clone this module:
```sh
$ git clone https://github.com/rogercoll/canopus.git
```
### 2. Building it:
```sh
$ cd canopus/
$ go build -o canopus
```
### 3. Run it:
```golang
./canopus help
```
